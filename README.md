<h1>Weather App </h1>
<p>This is a Responsive Weather Application is built using REACT, showcasing current weather conditions and 5-day forecasts for searched cities.</p>
<p>Technologies Used : </p>
<ul>
 <li>React</li>
 <li>Tailwind CSS</li>
 <li>OpenWeatherMap API</li>
</ul>
